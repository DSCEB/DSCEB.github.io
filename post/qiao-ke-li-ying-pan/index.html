巧克力硬盘

2020-10-22 工业设计

巧克力硬盘是一个像巧克力一样，可以“掰开”分享给他人使用的移动硬盘。它具有模块化功能，无论在虚拟空间划分或者物理存储位置，都可以根据使用者的需求进行归纳。当有他人需要使用硬盘里的一个文件时，只需要将该文件所储模块“掰开”当作U盘给他人使用即可，无需将整个硬盘交给他人使用，更加方便、安全。
ChocoDisk is a Mobile Hard Disk like chocolate that can be "broken" and shared with others. It has a modular function, regardless of whether it is divided in virtual space or physical storage location, it can be summarized according to the needs of users. When someone needs to use a file in the hard disk, you only need to "open" the module stored in the file and use it as a U Disk for others to use. There is no need to give the entire hard disk to others for use, which is more convenient and safer.

Design starts from change and ends in balance.
